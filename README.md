### 自定义Dialog，MixDialog有三个功能
###### 1.确认取消弹框，使用方式：
```
MixDialog.getMixDialog().showConfirmDialog(this,"标题","内容",new OnConfirmDialogListener(){
		@Override
        public void OnConfirm() {
            Log.e("=====","OnConfirm");
        }

        @Override
        public void OnCancel() {
            Log.e("=====","OnCancel");
        }
})
```
